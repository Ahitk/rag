Anonyme Anrufe
Anrufer, die keine Rufnummer übermitteln (anonyme Anrufer), können Sie sperren. Sie können entweder alle anonymen Anrufer sperren oder nur anonyme Anrufer, die eine bestimmte Rufnummer angerufen haben:
Klicken Sie in der Benutzeroberfläche der FRITZ!Box auf "Telefonie".
Klicken Sie im Menü "Telefonie" auf "Rufbehandlung".
Klicken Sie auf die Registerkarte "Rufsperren".
Klicken Sie im Abschnitt "Rufnummernbereiche sperren" auf die Schaltfläche "Bereich hinzufügen".
Wählen Sie in der Ausklappliste "Bereich" den Eintrag "Ohne Rufnummer (Anonym)" aus.
Wählen Sie aus, ob nur Anrufe an eine bestimmte Rufnummer oder alle Anrufe gesperrt werden sollen.
Klicken Sie zum Speichern der Einstellungen auf "Übernehmen".
Anrufe von bestimmter Rufnummer oder aus einem RufnummernbereichKlicken Sie in der Benutzeroberfläche der FRITZ!Box auf "Telefonie".
Klicken Sie im Menü "Telefonie" auf "Rufbehandlung".
Klicken Sie auf die Registerkarte "Rufsperren".
Falls Sie eine bestimmte Rufnummer sperren möchten:
Klicken Sie auf die Schaltfläche "Rufnummer hinzufügen".
Tragen Sie die Rufnummer und einen Namen ein. Der Name wird dann in der Anrufliste der FRITZ!Box angezeigt.
Klicken Sie zum Speichern der Einstellungen auf "Übernehmen".
 
Falls Sie einen Rufnummernbereich sperren möchten:
Klicken Sie auf die Schaltfläche "Bereich hinzufügen".
Wählen Sie in der Ausklappliste "Bereich" den Eintrag "Rufnummernbereich" aus.
Wählen Sie aus, ob nur Anrufe an eine bestimmte Rufnummer oder alle Anrufe gesperrt werden sollen.
Tragen Sie in das Feld "Rufnummernbereich" einen beliebigen Bereich (z.B. 0900 oder 123) ein.
Tragen Sie einen Namen für die Rufsperre ein. Der Name wird in der Anrufliste der FRITZ!Box angezeigt.
Klicken Sie zum Speichern der Einstellungen auf "Übernehmen".Gruß Marcel
