Question: Wie erhalte ich die Software von dem Telekom Sicherheitspaket?
Answer: Sie erhalten nach der Bestellung des Sicherheitspakets eine Eingangsbestätigung per E-Mail. In dieser ist der Link zumHerunterladender aktuellen Software von Norton enthalten.
Sofern Sie das Sicherheitspaket über Ihren Mobilfunk-Vertrag gebucht haben, erhalten Sie eine SMS mit dem Downloadlink. Alternativ können Sie die Software über dasKundencenterper Klick auf "Details" bei Ihrem Sicherheitspaket herunterladen.
