Subject: eSim
Content: Hallo!Ich würde gerne einen eSim Code für eine vorhandene Nummer eines Telekom Mobilfunkvertrag beantragen.Nun lese ich, dass das nur für XL Tarife geht.Ist das so? Muss ich einen XL Tarif besitzen um einen eSim Code zu erhalten?Danke und einen schönen Gruß!Michael

Accepted Solutions:
- Beim XL Tarif gibt es eine vergünstigte MultiSIM für die Watch für monatlich 4,95 €, während eine normale MultiSIM monatlich 29,95 € kostet. Bei den niedrigeren MagentaMobil Tarifen kosten alle MultiSIM 4,95 € im Monat.Beim Einrichten der Watch auf dem iPhone wird das Buchen einer MultiSIM angeboten.
