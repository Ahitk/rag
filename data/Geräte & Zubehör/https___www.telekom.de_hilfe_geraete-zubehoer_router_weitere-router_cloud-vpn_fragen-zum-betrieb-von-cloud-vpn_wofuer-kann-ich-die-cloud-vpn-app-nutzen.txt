Question: Wofür kann ich die Cloud VPN App nutzen?
Answer: Die Cloud VPN App ist die zentrale Steuerzentrale für Ihr Cloud VPN.

Sie bietet  eine grafische Benutzeroberfläche zur Darstellung der Datenströme im Cloud VPN als Basis für das Visual QoS.

Weiterhin erhalten Sie umfangreiche Möglichkeiten zur Eingabe und Einstellung von Konfigurations-Parametern für Ihr Cloud VPN.

Installieren Sie jetzt die Cloud VPN App für Ihr iPad.(Sie werden weitergeleitet auf itunes.apple.com)
