Question: Wie kann ich die Begrüßung auf meiner Telekom SprachBox ändern?
Answer: So ändern Sie die Begrüßung auf Ihrer SprachBox:
• Voicemail AppInstallieren Sie dieVoicemail Appund melden Sie sich mit dem Telekom Login an. Unter dem Menü-Punkt "Begrüßung" können Sie einen fertigen Text auswählen oder über das "+" eine eigene Ansage hinzufügen.
• TelefonischWählen Sie von Ihrem Festnetz-Anschluss die kostenfreie Rufnummer 0800 33 02424. Drücken Sie im Hauptmenü die Tastenkombination "311". Sprechen Sie Ihren gewünschten Text auf.
