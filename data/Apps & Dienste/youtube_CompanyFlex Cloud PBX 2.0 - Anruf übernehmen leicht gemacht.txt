Question:
Haben Sie weitere Fragen zur Funktion "Hof übernehmen", wie Sie ein Gespräch von Ihrem Desktop auf Ihr Mobiltelefon übertragen können?

Answer:
Hallo, mein Name ist Markus und ich habe heute eine interessante Funktion von Cloud bewegts vorgestellt. Es geht um die Übernahme von Anrufen, bei der Sie ein bestehendes Gespräch von Ihrem Desktop zu Ihrem Mobilgerät übertragen können. Bevor Sie diese Funktion benutzen können, müssen Sie sicherstellen, dass die Geräte Ihnen zugewiesen sind. Wenn ein Gerät fehlt, können Sie es einfach hinzufügen. Um ein Gespräch von Ihrem Desktop zu Ihrem Handy zu überführen, wählen Sie in der App das Telefonsymbol aus und drücken auf das Pluszeichen. Wählen Sie die Funktion 'An Hof übertragen', und das Gespräch wird auf Ihr Mobilgerät übertragen und auf dem Desktop beendet. Dieser Prozess kann auch umgekehrt ausgeführt werden.
