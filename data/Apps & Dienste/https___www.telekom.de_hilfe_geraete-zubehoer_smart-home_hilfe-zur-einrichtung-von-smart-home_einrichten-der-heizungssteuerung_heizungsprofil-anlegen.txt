Question: Wie kann ich in Magenta SmartHome ein Heizungsprofil mit Zeitspannen anlegen und ändern?
Answer: Ein neues Heizungsprofil erstellenÜber den Stift rechts oben in der Leiste können Sie ein Heizungsprofil bearbeiten und über das "+ Profil hinzufügen" legen Sie ein neues Heizungsprofil an. Das neue Heizungsprofil übernimmt alle Temperaturen Ihres aktiven Profils. Diese können Sie im Anschluss verändern, neue Temperaturen hinzufügen oder kopieren.
Eine Zeitspanne hinzufügen oder ändernWählen Sie in der Heizungssteuerung ein Heizungsprofil aus und tippen auf den Stift rechts oben. Tippen Sie dann erneut auf das Heizungsprofil und das "+" oben rechts, um eine neue Zeitspanne hinzuzufügen.Sobald die Einstellungen erscheinen, drücken Sie auf "Beginn" bzw. "Ende", um mithilfe des Einstellrades die Zeitspanne zu definieren. Verschieben Sie den Balken oder drücken Sie auf eine Ihrer Grundtemperaturen am linken Rand, um die Temperatur anzupassen.Drücken Sie anschließend auf "Speichern". So gelangen Sie zurück zur vorherigen Ansicht. Die Änderungen sind jetzt für Ihr Heizungsprofil wirksam.
Einen Tagesablauf auf weitere Räume übertragenWählen Sie im Heizungsmenü den Raum, aus dessen Einstellungen Sie kopieren wollen. Klicken Sie unten auf "Raum kopieren". Wählen Sie im nächsten Schritt den Raum aus, auf den Sie den Wochenablauf übertragen möchten.
Eine Zeitspanne löschenWenn Sie eine Zeitspanne aus der Heizungssteuerung entfernen, wird dadurch die vorherige Zeitspanne verlängert. Das bedeutet, dass die der vorherigen Zeitspanne zugeordnete Solltemperatur bis zum Endzeitpunkt der gelöschten Zeitspanne beibehalten wird.
Ein Heizungsprofil wechselnWechseln Sie das Heizungsprofil, dann werden die im gewählten Profil eingestellten Temperaturen sofort an Ihre Heizkörperthermostate und wenn vorhanden – an das Wandthermostat übertragen.
Ein Heizungsprofil löschenÜber den Stift rechts oben in der Leiste gelangen Sie zur Bearbeitung von Heizungsprofilen. Für das Löschen nutzen Sie den Mülleimer rechts neben dem betreffenden Profil, tippen anschließend auf den Haken rechts oben in der Leiste, dann wird das Profil unwiderruflich gelöscht.Hinweis:Die Profile "Komfort" und "Unterwegs" können geändert, aber nicht gelöscht werden. Aktive Profile können ebenfalls nicht gelöscht werden. Aktivieren Sie ein anderes Profil und löschen dann das gewünschte.
