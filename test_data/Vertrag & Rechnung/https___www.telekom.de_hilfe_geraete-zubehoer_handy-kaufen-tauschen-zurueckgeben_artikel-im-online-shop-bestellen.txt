Question: Wie kann ich einen Artikel im Online-Shop bestellen?
Answer: Auf unseren Online-Shop-Seiten unter der u.a. Internetadresse können Sie sich ausführlich über die einzelnen Handys informieren. Ganz egal, ob Sie bereits Kunde sind oder Neukunde werden wollen - Sie können unsere Artikel ganz einfach in sieben Schritten bestellen:
1. Warenkorb
In dieser Liste stellen Sie ganz unverbindlich die Produkte zusammen, für die Sie sich interessieren.
2. Tarifwahl
Hier wählen Sie den für Sie günstigsten Tarif. Links führen zu den Tarifinformationen, ohne dass dabei der Bestellvorgang abgebrochen wird.
3. Zubehör und Wunschoptionen
Das passende Zubehör finden Sie hier auch, ebenso unsere Wunschoptionen, die Ihren gewählten Tarif für Sie noch individueller gestalten.
4. Vertragsdaten
In diesem Online-Formular geben Sie alle Daten ein, die für den Vertragsabschluss benötigt werden. Auch wenn Sie im Bestellvorgang einen Schritt zurückgehen, bleiben diese Daten erhalten.
5. Es folgt eine kurze Zusammenfassung von Datenschutzerklärungen, AGB, Rückgaberechten und weiteren Vertragsbedingungen.
Bitte bestätigen Sie diese per Klick. Bevor Sie die Bestellung dann abschicken, erhalten Sie eine Darstellung Ihrer Eingaben, die Sie einfach korrigieren können. Erst wenn Sie diese Angaben bestätigen, ist der Bestellvorgang abgeschlossen.
6. Zusammenfassung
Bevor Sie die Bestellung verbindlich abschicken, erhalten Sie eine Darstellung Ihrer Eingaben, die Sie einfach korrigieren können. Erst wenn Sie diese Angaben bestätigen, ist der Bestellvorgang abgeschlossen.
7. Bestätigung
Zuletzt erhalten Sie eine Bestätigung mit Bestelldaten und Auftragsnummer zum Ausdrucken. Zusätzlich schicken wir Ihnen diese Daten auch per E-Mail.
